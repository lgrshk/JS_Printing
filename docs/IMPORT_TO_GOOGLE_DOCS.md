# Інструкція: Імпорт документа в Google Docs

## Варіант 1: Завантажити Word файл в Google Docs (рекомендовано)

1. **Відкрийте Google Drive**: https://drive.google.com
2. **Натисніть "Створити"** → **"Завантажити файл"**
3. **Виберіть файл**: `docs/printing-architecture-comparison.docx`
4. **Після завантаження** → **Клікніть правою кнопкою на файл** → **"Відкрити за допомогою"** → **"Google Документи"**
5. Документ автоматично відкриється в Google Docs і буде доступний для редагування онлайн

## Варіант 2: Прямий імпорт в Google Docs

1. **Відкрийте Google Docs**: https://docs.google.com
2. **Натисніть "Створити"** → **"Завантажити файл"**
3. **Виберіть файл**: `docs/printing-architecture-comparison.docx`
4. Документ автоматично конвертується та відкриється в Google Docs

## Варіант 3: Копіювання тексту з Markdown

1. **Відкрийте файл**: `docs/printing-architecture-comparison.md`
2. **Скопіюйте весь вміст** (Cmd+A, Cmd+C)
3. **Відкрийте Google Docs**: https://docs.google.com
4. **Створіть новий документ** → **Вставте текст** (Cmd+V)
5. **Відформатуйте вручну** (заголовки, таблиці, списки)

## Варіант 4: Використання онлайн конвертерів

### Markdown to Google Docs конвертери:
- **CloudConvert**: https://cloudconvert.com/md-to-docx
  1. Завантажте `printing-architecture-comparison.md`
  2. Конвертуйте в DOCX
  3. Завантажте результат в Google Drive
  4. Відкрийте через Google Docs

- **Dillinger**: https://dillinger.io/
  1. Відкрийте файл Markdown
  2. Експортуйте як HTML
  3. Скопіюйте HTML в Google Docs
